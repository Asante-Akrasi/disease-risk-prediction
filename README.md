# Disease Risk Prediction (Diabetes)

## Overview
This project uses machine learning to predict the likelihood of diabetes based on medical data.

## Models Used
- Logistic Regression
- Random Forest

## Process
- Data cleaning
- Exploratory Data Analysis (EDA)
- Model training and evaluation

## Key Insights
- Glucose is the strongest predictor of diabetes
- BMI and age also contribute significantly
- Dataset is slightly imbalanced

## Results
Random Forest outperformed Logistic Regression in prediction accuracy.

## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Motivation
This project reflects my interest in applying AI to healthcare systems and improving early disease detection.
